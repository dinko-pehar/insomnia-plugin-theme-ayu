## Installation

- Go to Insomnia Settings > Plugins
- Type `insomnia-plugin-theme-ayu` and install the plugin

## Activating theme

1. Go to Application and select **Preferences**
2. Click on **Themes**
3. Select Ayu theme *(Light, Dark or Mirage)* to apply it
